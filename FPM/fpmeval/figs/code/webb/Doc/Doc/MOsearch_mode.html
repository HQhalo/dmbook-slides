Search mode

Magnum Opus supports five search modes for association discovery, search by leverage, lift, strength, coverage, and support. Two search modes are provided for itemset discovery, search by leverage and coverage. The only difference between these modes is that Magnum Opus finds the rules or itemsets with the highest value of the respective measure of interestingness in each mode.  For example, when Search Mode is set to Leverage, Magnum Opus finds the associations with the highest leverage, and when Search Mode is set to Strength, Magnum Opus finds the associations with the highest strength.

Under any search mode, Magnum Opus finds the number of rules or itemsets specified by the user, guaranteeing that those found are those with the highest values on the specified measure out of those that satisfy the user specified constraints. Magnum Opus will only find fewer than the specified number of associations if the search is terminated by the user or there are fewer than the specified number that satisfy the user specified constraints.

The associations that are found are listed in the output file in descending order on the measure specified by the current search mode.
