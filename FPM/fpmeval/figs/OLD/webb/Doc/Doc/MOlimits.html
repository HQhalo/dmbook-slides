Limits

Magnum Opus has the following limitations:

  • Names are limited to a maximum of 256 significant characters. Longer names are accepted, but two names that differ only beyond the 256th character will be treated as the same. This limit applies to the names of attributes, attribute-values, identifiers and items.
  • The number of cases is limited to a maximum of 2,147,483,647.
  • For attribute-value data, the number of attributes and the number of attribute-values are both limited to a maximum of 2,147,483,647.
  • For transaction data, the number of items is limited to a maximum of 2,147,483,647.
  • For identifier-item data, the number of transaction identifiers is also limited to a maximum of 2,147,483,647.
In practice the amount of available memory is more likely to constrain the operation of the system than are these built-in limits.
