Sampling

While Magnum Opus is designed for analyzing large data sets, when the data set is too large to be retained in memory, processing is dramatically slowed.

The proportion option instructs Magnum Opus to sample the data as it is imported. The argument to this option is a number between 0.0 and 1.0. This indicates the proportion of cases in the data that should be randomly included in the analysis. For example, mocl item-list-file=tutorial.itl proportion=0.1 will include in the analysis 0.1 (10%) of the cases in the data.

If the out-of-sample-holdout-evaluation command line argument is specified, the data that is not included in the sample will be used for holdout evaluation.
