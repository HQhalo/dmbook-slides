Data file

A data file is a text file that lists the cases to be analyzed in attribute-value data analysis.  Cases are listed one per line. For each case, a value must be provided for each attribute. The attribute values for a case must be listed in the order that they appear in the names file. Values must be separated by a delimiter. In the interactive version the user specifies the delimiter in the data import wizard.  In the command line version the user can specify the delimiter using the delimiter command.

The special attribute value `?’ indicates that no value is available for the attribute with respect to the current case. Missing values may also be indicated by an empty field. For example, both of the following lines specify cases for which values are missing for all but the second attribute, using comma as the delimiter.

,2,,

?,2,?,?
