Case

Cases are the basic unit of analysis used by Magnum Opus.  A case represents a single entity. Examples of the type of thing that might be used as a case include customers, transactions, and states of a dynamic process.

For attribute-value data, each case is described by a set of values for a number of attributes. The same attributes are used to describe all cases for a single analysis. The attributes for the analysis are defined in a names file. The cases are described in a data file.

For transaction analysis, each case is a transaction consisting of a number of related items.
